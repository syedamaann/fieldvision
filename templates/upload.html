About FieldVision

This project aims to develop monitoring tools for users using data collected from sensors installed in fields and applying Time Series Analysis to it. The tool will be web-based and will be available in local languages. This project aims to help users make informed decisions by providing them with useful statistical information about soil characteristics using real-time data from their fields..
